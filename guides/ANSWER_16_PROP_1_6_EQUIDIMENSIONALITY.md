# Answer to Q16: Equidimensionality Route for Proposition 1.6

## Preserved question

`prop_1_6` in `BEI/CohenMacaulay.lean` states:

> if `G` is connected, closed, and satisfies `SatisfiesProp1_6Condition`, then
> `S / J_G` is Cohen–Macaulay.

The question asked whether this could be proved directly using the local
equidimensionality definition of Cohen–Macaulayness by showing that all minimal primes
`P_S(G)` have the same quotient dimension, i.e. that for every minimal-prime set `S`,

`Fintype.card V - S.card + componentCount G S = Fintype.card V + componentCount G ∅`.

Equivalently, for connected `G`, whether one can show

`componentCount G S = S.card + 1`

for every `S` giving a minimal prime.

## Short answer

This is a plausible route, but it should **not** be the default next step unless it is
explicitly chosen as the branch to pursue.

Why:

1. it is not the route used in the paper;
2. it risks turning Proposition 1.6 into a new graph-combinatorics project;
3. the current repo is already much closer to the paper’s reduction through the initial
   ideal and the Herzog–Hibi bipartite graph `Γ`.

So the right recommendation is:

- treat the direct equidimensionality statement as a legitimate fallback route;
- do **not** let it displace the paper-faithful reduction as the main plan without a
  clear reason.

Update since the question was asked:

- the paper-faithful route has now moved forward via `prop_1_6_herzogHibi`, which
  packages the Herzog–Hibi graph conditions on the BEI side;
- that makes the direct equidimensionality route even more clearly a fallback, not the
  default next step.

## What the paper actually does

The proof in `BEI.tex` goes:

1. identify `in_<(J_G)` with a monomial ideal generated by `x_i y_j`;
2. transport it by the `y_j ↦ y_{j-1}` automorphism;
3. view the result as the edge ideal of a bipartite graph `Γ`;
4. check the Herzog–Hibi conditions (i), (ii), (iii) for `Γ`;
5. conclude `S'/I(Γ)` is Cohen–Macaulay;
6. transfer CM back to `S / J_G`.

So the paper does **not** prove Proposition 1.6 by a direct formula for
`componentCount G S` on minimal-prime sets.

## What is already available in Lean

The direct-equidimensionality route became more realistic because the repo now has:

- `complete_is_CM` in `BEI/CohenMacaulay.lean`
- `path_is_CM` in `BEI/PrimeDecompositionDimension.lean`
- `isCohenMacaulay_of_equidim_minimalPrimes`
- `ringKrullDim_quot_primeComponent`
- `minimalPrimes_characterization`
- `corollary_3_9`

This is enough to formulate the exact equidimensionality target cleanly.

## Why the path proof does not automatically generalize

For the path graph, the proof succeeds because the minimal-prime sets are controlled by
very rigid interval combinatorics:

- removing a valid set creates exactly `|S| + 1` components;
- the upper bound `c(S) ≤ |S| + 1` is easy;
- the lower bound comes from a strong induction using cut-vertex preservation under
  erasure.

For a general connected closed graph satisfying Proposition 1.6, none of this is yet
available in the repo:

- there is no Lean theorem classifying the valid minimal-prime sets;
- there is no current analogue of the path-specific upper/lower bound package;
- and the paper itself does not give such a direct argument.

So if this route works, it will require a genuinely new graph theorem.

## Recommended decision rule

Use this rule:

### Prefer the paper-faithful route if possible

If the reduction to the Herzog–Hibi bipartite graph `Γ` can be made formal with
reasonable effort, that should remain the main route for Proposition 1.6.

### Use the direct equidimensionality route only if it localizes cleanly

Only pursue the direct route if you can isolate a sharp graph statement of the form:

```text
For connected closed graphs satisfying SatisfiesProp1_6Condition,
every minimal-prime set S satisfies componentCount G S = S.card + 1.
```

and then break that into a short, self-contained combinatorial program.

If you cannot quickly identify such a program, do not continue pushing this route by
exploratory theorem hacking.

## Best next step if this route is chosen

Do **not** try to prove the full equality for all minimal-prime sets in one theorem.

Instead, split it into:

1. a theorem describing the shape of minimal-prime sets `S` under the Proposition 1.6
   hypotheses;
2. an upper bound `componentCount G S ≤ S.card + 1`;
3. a lower bound `componentCount G S ≥ S.card + 1`;
4. a dimension corollary for `ringKrullDim (R ⧸ P_S(G))`.

If step 1 does not stabilize, stop and return to the paper-faithful route.
