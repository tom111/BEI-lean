# Supplementary Code

This directory contains Lean files that are kept in the repository but are not part of
the main Herzog–Hibi–Hreinsdóttir–Kahle–Rauh (2010) paper formalization path.

Current contents:

- `RauhApproach.lean`: an archived alternative proof attempt for Theorem 2.1, based on
  Rauh (2013). It is not imported by `BEI.lean` and does not contribute to the main
  theorem-status accounting for the paper.
