## Proof that \(\mathcal{G}\) is a reduced Gröbner basis for \(J_G\)

### Notation and setup
- \(G\): simple graph on vertex set \([n] = \{1,\dots,n\}\).
- \(J_G\): binomial edge ideal generated by \(f_{ij} = x_i y_j - x_j y_i\) for edges \(\{i,j\} \in E(G)\) with \(i < j\).
- Monomial order \(<\) from Theorem 1.1: \(\operatorname{in}_<(f_{ij}) = x_i y_j\) for \(i < j\).
- Admissible path \(\pi: i = i_0, i_1, \dots, i_r = j\):
  1. Vertices distinct.
  2. For each internal vertex \(i_k\) (\(1 \le k \le r-1\)): \(i_k < i\) or \(i_k > j\).
  3. No proper subset of internal vertices yields a path from \(i\) to \(j\).
- Associated monomial: \(u_\pi = \left( \prod_{i_k > j} x_{i_k} \right) \left( \prod_{i_\ell < i} y_{i_\ell} \right)\).
- \(\mathcal{G} = \bigcup_{i<j} \{ u_\pi f_{ij} : \pi \text{ admissible from } i \text{ to } j \}\).

---

### Step 1: \(\mathcal{G} \subset J_G\)
Show \(u_\pi f_{ij} \in J_G\) for every admissible path \(\pi\) from \(i\) to \(j\) (\(i < j\)).  
Induction on path length \(r\) (number of edges).

**Base case \(r=1\)**: \(\pi\) is just edge \(\{i,j\}\). Then \(u_\pi = 1\), so \(u_\pi f_{ij} = f_{ij} \in J_G\).

**Inductive step \(r > 1\)**: Let \(\pi: i=i_0, i_1,\dots,i_{r-1},i_r=j\).  
Define \(A = \{ i_k : i_k < i \}\) and \(B = \{ i_\ell : i_\ell > j \}\).  
Since \(r>1\), \(A \cup B \neq \emptyset\). We consider two cases.

#### Case 1: \(A \neq \emptyset\)
Let \(i_{k_0} = \max A\). Split \(\pi\) at \(i_{k_0}\):
- \(\pi_1\): path from \(i_{k_0}\) to \(i\) along \(\pi\) (reverse direction).
- \(\pi_2\): path from \(i_{k_0}\) to \(j\) along \(\pi\).

Both \(\pi_1, \pi_2\) are admissible (check conditions using maximality of \(i_{k_0}\) and admissibility of \(\pi\)).  
By induction, \(g_1 = u_{\pi_1} f_{i_{k_0} i} \in J_G\) and \(g_2 = u_{\pi_2} f_{i_{k_0} j} \in J_G\).

Compute S-polynomial:
\[
\begin{aligned}
S(g_1, g_2) &= \frac{\operatorname{lcm}(\operatorname{in}_<(g_1), \operatorname{in}_<(g_2))}{\operatorname{in}_<(g_1)} g_1 - \frac{\operatorname{lcm}(\operatorname{in}_<(g_1), \operatorname{in}_<(g_2))}{\operatorname{in}_<(g_2)} g_2 \\
&= -\underbrace{y_{i_{k_0}} u_{\pi_1} u_{\pi_2}}_{= u_\pi} f_{ij} = -u_\pi f_{ij}.
\end{aligned}
\]
Thus \(u_\pi f_{ij} = -S(g_1,g_2) \in J_G\).

#### Case 2: \(B \neq \emptyset\)
Symmetric, using \(i_{\ell_0} = \min B\).

By induction, \(\mathcal{G} \subset J_G\).

---

### Step 2: \(\mathcal{G}\) is a Gröbner basis
Use Buchberger's criterion: show all S-polynomials reduce to \(0\) modulo \(\mathcal{G}\).  
Take \(g_1 = u_\pi f_{ij}\), \(g_2 = u_\sigma f_{k\ell}\) with \(i<j\), \(k<\ell\).  
Case analysis based on \(\{i,j\} \cap \{k,\ell\}\).

#### Case 1: \(i=k\), \(j=\ell\)
Then \(S(g_1,g_2) = 0\) because \(\operatorname{lcm}(u_\pi x_i y_j, u_\sigma x_i y_j) = \operatorname{lcm}(u_\pi, u_\sigma) x_i y_j\) and the binomials cancel.

#### Case 2: \(\{i,j\} \cap \{k,\ell\} = \emptyset\), or \(i=\ell\), or \(k=j\)
Here \(\operatorname{in}_<(f_{ij}) = x_i y_j\) and \(\operatorname{in}_<(f_{k\ell}) = x_k y_\ell\) form a regular sequence.  
A general lemma: if \(\operatorname{in}_<(f)\) and \(\operatorname{in}_<(g)\) form a regular sequence, then for any monomials \(u,v\), \(S(uf, vg)\) reduces to \(0\).  
Thus \(S(g_1,g_2)\) reduces to \(0\).

#### Case 3: \(i = k\), \(j \neq \ell\) (the case \(i \neq k\), \(j = \ell\) is symmetric)
Assume \(j < \ell\). Write \(\pi: i=i_0,\dots,i_r=j\), \(\sigma: i=i'_0,\dots,i'_s=\ell\).  
Let \(a,b\) be indices with \(i_a = i'_b\) such that \(\{i_{a+1},\dots,i_r\} \cap \{i'_{b+1},\dots,i'_s\} = \emptyset\).  
Construct path \(\tau\) from \(j\) to \(\ell\):
\[
\tau: j = i_r, i_{r-1}, \dots, i_{a+1}, i_a = i'_b, i'_{b+1}, \dots, i'_s = \ell.
\]
Relabel \(\tau: j = j_0, j_1, \dots, j_t = \ell\).

Define indices \(0 = t(0) < t(1) < \dots < t(q) = t\) where:
- \(t(1) = \min\{ c : j_c > j \}\),
- \(t(c) = \min\{ d > t(c-1) : j_d > j_{t(c-1)} \}\) for \(c \ge 1\), until \(j_{t(q)} = \ell\).

Each subpath \(\tau_c: j_{t(c-1)}, \dots, j_{t(c)}\) is admissible.  
Let \(w = y_i \operatorname{lcm}(u_\pi, u_\sigma)\). Then:
\[
S(g_1, g_2) = -w f_{j\ell}.
\]
We exhibit a standard expression:
\[
w f_{j\ell} = \sum_{c=1}^q v_{\tau_c} u_{\tau_c} f_{j_{t(c-1)} j_{t(c)}},
\]
where
\[
v_{\tau_c} = 
\begin{cases}
\dfrac{x_\ell w}{u_{\tau_1} x_{j_{t(1)}}}, & c=1,\\[8pt]
\dfrac{x_j x_\ell w}{u_{\tau_c} x_{j_{t(c-1)}} x_{j_{t(c)}}}, & 1 < c < q,\\[8pt]
\dfrac{x_j w}{u_{\tau_q} x_{j_{t(q-1)}}}, & c=q.
\end{cases}
\]
Each \(v_{\tau_c}\) is a monomial because \(x_{j_{t(c)}}\) divides \(w\) (since \(j_{t(c)}\) is an internal vertex of \(\pi\) with \(j_{t(c)} > j\)).  
The identity holds by algebraic manipulation (telescoping sum).  
The leading terms satisfy:
\[
\operatorname{in}_<(w f_{j\ell}) = w x_j y_\ell > \dots > \operatorname{in}_<(v_{\tau_1} u_{\tau_1} f_{j j_{t(1)}}),
\]
so the expression is standard with remainder \(0\). Hence \(S(g_1,g_2)\) reduces to \(0\).

By Buchberger, \(\mathcal{G}\) is a Gröbner basis.

---

### Step 3: \(\mathcal{G}\) is reduced
Need: for distinct \(g_1 = u_\pi f_{ij}\), \(g_2 = u_\sigma f_{k\ell}\) in \(\mathcal{G}\), \(\operatorname{in}_<(g_1) = u_\pi x_i y_j\) divides **no** term of \(g_2\) (i.e., neither \(u_\sigma x_k y_\ell\) nor \(u_\sigma x_\ell y_k\)).

#### Case 1: \(i=k\), \(j=\ell\)
If \(u_\pi x_i y_j \mid u_\sigma x_i y_j\), then \(u_\pi \mid u_\sigma\). This implies the internal vertices of \(\pi\) are a proper subset of those of \(\sigma\), contradicting admissibility of \(\sigma\) (condition (iii)).  
If \(u_\pi x_i y_j \mid u_\sigma x_j y_i\), then \(x_i y_j \mid x_j y_i\), impossible since \(i<j\).

#### Case 2: \(i=k\), \(j \neq \ell\)
If \(u_\pi x_i y_j \mid u_\sigma x_i y_\ell\), then \(y_j \mid u_\sigma\) so \(j\) is an internal vertex of \(\sigma\) with \(j < i\), contradicting \(i<j\).  
If \(u_\pi x_i y_j \mid u_\sigma x_\ell y_i\), then \(x_i \mid u_\sigma x_\ell\) forces \(i = \ell\), but then \(i = \ell < j\) and also \(i < \ell\) from \(\sigma\), contradiction.

#### Case 3: \(\{i,j\} \cap \{k,\ell\} = \emptyset\)
If \(u_\pi x_i y_j \mid u_\sigma x_k y_\ell\), then \(x_i \mid u_\sigma\) and \(y_j \mid u_\sigma\), so \(i > \ell\) and \(j < k\). With \(i<j\) and \(k<\ell\), we get \(i > \ell > k > j > i\), contradiction.  
Division by the other term yields similar contradiction.

Thus \(\mathcal{G}\) is reduced.

---

### Conclusion
\(\mathcal{G} \subset J_G\), \(\mathcal{G}\) is a Gröbner basis (by Buchberger), and reduced. Hence \(\mathcal{G}\) is a reduced Gröbner basis of \(J_G\).
